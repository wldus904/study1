1. 학생 페이지

 - 자기 성적 확인 (이번 시험이든 과거 시험이든 검색해서)
 - (성적 이의 제기)
 - 자기 기본 정보 확인

2. 교직원 페이지 (기본정보 DB 박혀있어)
 - 학생 검색 페이지 : 학생 정보, 학급정보(현재), 담당 선생님 정보가 리스트로 검색
 - 학생 등록 페이지 : 기본 정보 등록
 - 학생 상세 페이지 : 학생 정보, 모든 기간 학급정보, 모든 성적 정보, (추가: 힉셍이 올린 이의제기 확인 - 게시판 + 대댓글)
 - 성적 검색 페이지 : 모든 학생의 성적 정보 (년도, 학기로 검색가능, 학생 정보로도 검색 가능)
-  성적 입력 페이지 : 과목별로 입력 가능, 년도 + 학기로 전체 학생 성적 입력가능
